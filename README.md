# FlowMo
FlowMo is a language that is as fast as C++ and as easy as FlowMo

- Compilation Procedure
  - First give permission to run bash files
    ```bash
    chmod a+x *
    ```
  
  - Then compile it
    ```bash
    ./build.sh
    ```
  Compilation done
  
  - Now change directory
  
    ```bash
    cd bin
    ```
  - Now run sample code
    ```python3
    -print Hello, World! ;
    -exec cowsay this is a exec command ;
    this is a comment
    ```
  
   - To run it  
  
      ```bash
     ./flowmo ./file.fm
       ```
  
  and done.
